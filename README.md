# Smart-Parking-System
Smart Parking System adalah sebuah prototipe sistem manajemen perparkiran otomatis yang dirancang untuk mengatasi permasalahan efisiensi lahan parkir di area gedung atau pusat perbelanjaan. Proyek ini memadukan perangkat keras (hardware) dan perangkat lunak (software) berbasis mikrokontroler untuk menciptakan ekosistem parkir yang cerdas.
